# tree-sitter-surrealdb

[SurrealDB](https://surrealdb.com) grammar for [tree-sitter](https://github.com/tree-sitter/tree-sitter)

## features

- [x] highlighting
- [x] tokenization
  - [x] comments
  - [ ] Data model
    - [x] records
    - [x] basic types
    - [x] strings
    - [x] numbers
    - [ ] datetimes tokenized as string
    - [x] durations
    - [x] futures
    - [x] casting
  - [x] Statements
    - [x] use
    - [x] let
    - [x] begin
    - [x] cancel
    - [x] commit
    - [x] if else
    - [x] select
    - [x] insert
    - [x] create
    - [x] update
    - [x] relate
    - [x] delete
    - [x] define
    - [x] remove
    - [x] info
    - [x] sleep
  - [x] operators
  - [x] parameters
  - [x] transactions
  - [x] functions
  - [x] scripting functions
